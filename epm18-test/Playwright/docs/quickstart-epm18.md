# epm18-test Playwright Quickstart

`powershell
cd C:\RegressionTesting\epm18-test\Playwright
npm install
npm run auth:refresh
npm run test:e2e:evidence:full
`

The default pod settings are in .pod-settings.json and .env.epm18_test.
Use playwright\.auth\user.epm18_test.json for the default storage-state file.
